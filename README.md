## Online Government Service Center

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/0b12080b4a184b9c9b70cad57f152531/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Document Query
5. Education and Training
6. Convenience Services
7. Communication Forum
8. Notice Announcements
9. Complaints and Suggestions
10. Backend Management
11. Online Customer Service
12. Personal Center
13. My Posts
14. My Favorites


**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. User Management
4. Document Query Management
5. Query Information Management
6. Education and Training Management
7. Certification Application Management
8. Convenience Services Management
9. Payment Services Management
10. Complaints and Suggestions
11. Administrator Management
12. Communication Forum
13. System Management (About Us, Notice Announcements, System Introduction, Online Customer Service, Carousel Management)
